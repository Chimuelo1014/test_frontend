# CrudCloud Frontend

Frontend de la plataforma CrudCloud - Gestión de instancias de bases de datos en la nube.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Docker

```bash
docker build -t crudcloud-frontend .
docker run -p 80:80 crudcloud-frontend
```

## Stack

- React 18
- Vite
- TailwindCSS
- React Router
- Lucide Icons
